2. Concurrency & Threads

CompletableFuture (Non-Blocking)

CompletableFuture future = CompletableFuture.supplyAsync(() -> fetchUserData())
    .thenApply(user -> user.toUpperCase())
    .exceptionally(ex -> "Error");
Virtual Threads (Java 21): Lightweight threads managed by the JVM. Use Executors.newVirtualThreadPerTaskExecutor() to handle millions of I/O bound tasks efficiently without OS thread limits.

3. JVM Memory Model

Heap: Objects and JRE classes (Shared across threads). Includes Eden Space, Survivor Spaces, and Tenured Generation.

Stack: Local variables and method calls (One per thread).

Metaspace: Class definitions and static variables (Off-heap).
