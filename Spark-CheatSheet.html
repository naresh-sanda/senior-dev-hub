1. Spark Architecture

Driver: Runs the main() method, creates SparkContext, and schedules tasks.

Executors: Worker nodes that execute the tasks and store data in memory or disk.

2. Transformations vs Actions

ConceptBehaviorExamples
TransformationsLazy evaluation. Builds the DAG but doesn't execute anything yet.map(), filter(), join(), groupBy()
ActionsTriggers actual execution and returns a result to the driver or writes to storage.count(), collect(), save(), show()

3. PySpark DataFrames

df = spark.read.csv("data.csv", header=True, inferSchema=True)

# Filter and Aggregate
result = df.filter(df.age > 30) \
           .groupBy("department") \
           .agg(F.avg("salary").alias("avg_salary"))
result.show()
