Kubernetes (K8s)

Author: Naresh Kumar Sanda

Pods, Deployments, Services, and Ingress routing.

1. Core Objects

ObjectDescription
PodSmallest deployable unit. Usually contains one container.
DeploymentManages Pods. Ensures X replicas are always running. Handles rolling updates.
ServiceStable IP address that load-balances traffic across multiple ephemeral Pods.
IngressAPI Gateway that routes external HTTP traffic to internal Services based on URL paths.

2. Deployment YAML

apiVersion: apps/v1
kind: Deployment
metadata:
  name: myapp-deployment
spec:
  replicas: 3
  selector:
    matchLabels:
      app: myapp
  template:
    metadata:
      labels:
        app: myapp
    spec:
      containers:
      - name: myapp
        image: myapp:latest
        ports:
        - containerPort: 8080

3. Kubectl Commands

kubectl apply -f deployment.yaml  # Deploy
kubectl get pods                  # View running pods
kubectl logs <pod-name>           # Check logs
kubectl scale deployment myapp --replicas=5
