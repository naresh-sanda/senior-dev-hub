Docker Containerization

Author: Naresh Kumar Sanda

Images vs Containers, Dockerfiles, and Docker Compose.

1. Images vs Containers

An Image is a read-only template (like a Class in Java). A Container is a runnable instance of an image (like an Object in Java).

2. The Dockerfile

FROM openjdk:17-jdk-slim
WORKDIR /app
COPY target/myapp.jar app.jar
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]

3. Essential CLI Commands

docker build -t myapp:latest .   # Build image
docker run -p 8080:8080 myapp    # Run container mapping port 8080
docker ps                        # List running containers
docker exec -it <id> /bin/sh     # SSH into a container
